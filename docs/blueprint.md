# **App Name**: DubaiNegoce

## Core Features:

- Product Catalog: Display a catalog of oriental perfumes with details like name, brand, type, fragrance family, notes, price, and stock.
- Product Filtering and Sorting: Allow users to filter perfumes by brand, gender, fragrance family, and price, and sort by popularity or newest arrivals.
- Product Detail Page: Show detailed information for each perfume, including photos, olfactory pyramid, description, and customer reviews.
- Shopping Cart and Checkout: Enable users to add perfumes to a shopping cart, proceed to checkout via Stripe, and complete their purchase.
- Customer Reviews: Allow customers to leave reviews and ratings for perfumes, stored in Firestore.
- Search Engine Optimization: Optimize product pages with relevant keywords, meta descriptions, and schema.org markup to improve search engine rankings.
- AI Product Description Generation: Use a generative AI tool to create original long product descriptions by combining details from a product CSV file with fragrance profiles found on external sites.

## Style Guidelines:

- Primary color: Gold (#D4AF37) to evoke luxury and Middle Eastern aesthetics.
- Background color: Dark charcoal (#1A202C) provides a sophisticated and rich backdrop, allowing the gold accents and product images to stand out.
- Accent color: Off-white (#FAF9F6) used for text and key UI elements to ensure legibility against the dark background.
- Headline font: 'Playfair', a serif font, adds elegance and high-end appeal.
- Body font: 'PT Sans' ensures readability and balances the sophisticated headlines, being a humanist sans-serif.
- Use simple, elegant line icons in gold to represent menu items, search, account, and cart functions.
- Employ a clean, spacious layout inspired by high-end perfume e-commerce sites, emphasizing product visuals.
- Incorporate subtle animations on hover effects and transitions to enhance user engagement and provide a premium feel.
